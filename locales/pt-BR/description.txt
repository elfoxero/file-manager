Gerenciador de Arquivos (File Manager) é um aplicativo para Firefox OS que permite explorar e gerenciar arquivos do cartão SD.

Requerimentos:

Este aplicativo requer permissões para acessar o cartão SD.

Características:

- Permite copiar, mover, renomear e excluír arquivos.
- Permite criar, renomear e excluír pastas.
- Abre qualquer arquivo como um arquivo de música, vídeo, texto ou imagem.
- Abre arquivos PDF (requer Firefox OS 1.4 ou superior).
- Permite definir qualquer imagem como papel de parede.
- Permite compartilhar qualquer arquivo via bluetooth, email, entre outros.
- Permite criar e editar arquivos simples.
- Permite gerenciar qualquer pasta com um toque longo.

Para desenvolvedores:

Se você desenvolveu um aplicativo que interage com arquivos no cartão SD, o Gerenciador de Arquivos (File Manager) pode usar seu aplicativo para abrir aquivos diretamente. Para mais informações, veja o repositório do Github.

Recursos:

Este aplicativo é aberto sob licença GPL.
Usa uma variante da biblioteca Building Blocks e tema de ícones de Numix Circle.

Este aplicativo foi testado no ZTE Open, com Firefox OS 1.1, e em simuladores de Firefox OS 1.3, 1.4 e 2.0.
