File Manager ist eine Anwendung für Firefox OS zum Untersuchen und Verwalten von Dateien auf der SD-Karte.

Anforderungen:

Diese Anwendung benötigt die Berechtigung zum Zugriff auf die SD-Karte.

Features:

- Ermöglicht das Kopieren, Verschieben, Umbenennen und Löschen von Dateien.
- Ermöglicht das Erstellen, Umbenennen und Löschen von Ordnern.
- Öffnet Musik-, Video-, Text- und Bilddateien.
- Öffnet PDF-Dateien (benötigt Firefox OS 1.4 oder höher).
- Öffnet eine beliebige Datei als Musik, Video, Text oder Bild.
- Ermöglicht es, ein beliebiges Bild als Hintergrund zu setzen.
- Ermöglicht das Teilen einer beliebigen Datei über Bluetooth, Email oder mehr.
- Ermöglicht das Erstellen und Bearbeiten von einfachen Textdateien.
- Ermöglicht das Verwalten eines Orders durch langen Druck.

Für Entwickler:

Wenn Sie selbst eine Anwendung entwickelt haben, die mit Dateien auf der SD-Karte interagiert, dann kann File Manager die Dateien direkt mir Ihrer Anwendung öffnen. Mehr Informationen dazu finden sich im Github-Repository.

Ressourcen:

Diese Anwendung ist Open Source und steht under der GPL3+ Lizenz.
Verwendet eine Variante der Bibliothek Building Blocks und das Icon Theme von Numix Circle.

Diese Anwendung wurde auf einem ZTE Open mit Firefox OS 1.1 sowie Simulatoren für Firefox OS 1.3, 1.4 und 2.0 getestet.
